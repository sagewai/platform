# Patterns — production reference

This is **Tier 4** of the [example catalogue](/docs/lighthouse) — reference-grade examples for harder problems. Each is 200-400 lines but lacks the audience-pin storytelling of a [Tier 5 lighthouse](/docs/lighthouse). Goal: *"here's a complete-ish solution to a non-trivial problem."*

These examples are organised by pillar. They're the pages you reach when you've outgrown a [foundation](/docs/foundation) example and you're not yet ready to read the lighthouse work.
